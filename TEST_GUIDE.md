# TeleportPlugin 测试流程

## 测试前准备

### 1. 服务器环境
- 启动 Minecraft 服务器
- 确保插件已加载（检查控制台无错误）
- 创建一个测试玩家账号或使用已有账号

### 2. 测试前清理
```
# 删除旧的插件数据（可选，用于测试全新安装）
/stop  # 停止服务器
# 删除 plugins/TeleportPlugin 文件夹
# 重启服务器
```

## 测试项目

### 模块 1：基础传送命令

| 步骤 | 测试内容 | 预期结果 | 状态 |
|------|----------|----------|------|
| 1 | `/spawn` | 传送到主城位置，显示成功消息 | ☐ |
| 2 | `/setspawn` | 设置主城到当前位置，显示成功消息 | ☐ |
| 3 | `/spawn` 再次 | 传送到新的主城位置 | ☐ |

| 步骤 | 测试内容 | 预期结果 | 状态 |
|------|----------|----------|------|
| 1 | `/rtp` | 随机传送到某位置 | ☐ |

| 3 | `/rtpinfo` | 显示免费次数和价格 | ☐ |

| 步骤 | 测试内容 | 预期结果 | 状态 |

### 模块 5：经济系统
